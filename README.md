# Learnix - Advanced AI-Powered Study Assistant

Learnix is a sophisticated study assistant that leverages advanced AI to generate high-quality educational assessments. The system uses comprehensive content analysis, intelligent question generation, and analytical grading to provide an exceptional learning experience.

## üß† Advanced AI Capabilities

### Comprehensive Content Analysis
- **Content Extraction & Prioritization**: Advanced algorithms extract core educational content while filtering out metadata, copyright information, and formatting artifacts
- **Complexity Assessment**: Multi-dimensional analysis of content complexity, conceptual depth, and theoretical sophistication
- **Cognitive Demand Analysis**: Identifies required cognitive skills and prerequisite knowledge
- **Question Diversity Potential**: Evaluates content's ability to generate diverse, high-quality questions

### Intelligent Question Generation
- **Concept-Focused Questions**: Generates questions that test deep understanding of core concepts, theories, and principles
- **Metadata Filtering**: Completely ignores copyright, edition, publisher, page numbers, and administrative details
- **Numerical Content Detection**: Automatically identifies mathematical elements and generates appropriate numerical questions
- **Quality Validation**: Comprehensive validation system ensures questions meet high educational standards

### Advanced Question Types
- **Conceptual Understanding**: Tests grasp of fundamental principles and definitions
- **Analysis & Interpretation**: Requires critical analysis of information and concepts
- **Application Questions**: Tests ability to apply knowledge to new scenarios
- **Comparison & Contrast**: Evaluates understanding of relationships between concepts
- **Problem-Solving**: Tests critical thinking and analytical reasoning
- **Synthesis & Evaluation**: Requires combining information and making judgments
- **Mathematical Reasoning**: Numerical calculations, formula applications, and data interpretation

### Sophisticated Grading System
- **Multi-Criteria Assessment**: Evaluates conceptual understanding, accuracy, completeness, clarity, and critical thinking
- **Partial Credit System**: Provides fair assessment for partially correct answers
- **Numerical Question Grading**: Specialized grading for mathematical problems with step-by-step evaluation
- **Analytical Feedback**: Detailed, constructive feedback that helps students learn and improve
- **Understanding-Focused**: Emphasizes comprehension over memorization

## üöÄ Features

### Multiple Choice Questions (MCQ)
- **Intelligent Distractors**: Plausible options based on common misconceptions
- **Quality Assurance**: All options are grammatically correct and similar in complexity
- **No Obvious Wrong Answers**: Avoids "all/none of the above" and obviously incorrect options
- **Mathematical Integration**: Includes calculation-based questions when content supports it

### Short Answer Questions (QA)
- **Deep Content Analysis**: Extracts and prioritizes educational content by importance
- **Comprehensive Grading**: AI-powered assessment with detailed feedback
- **Partial Credit**: Rewards partially correct answers and approaches
- **Numerical Problem Solving**: Step-by-step calculation problems with mathematical accuracy

### Content Processing Pipeline
1. **Content Extraction**: Removes metadata and focuses on educational content
2. **Numerical Detection**: Identifies mathematical elements and calculation opportunities
3. **Complexity Analysis**: Assesses content depth and educational value
4. **Question Generation**: Creates diverse, high-quality questions
5. **Quality Validation**: Ensures questions meet educational standards
6. **Grading & Feedback**: Provides comprehensive assessment and learning guidance

## üöÄ Getting Started

### Prerequisites
- Python 3.8+
- Node.js 16+
- Google Gemini API key

### Installation
1. Clone the repository
2. Install backend dependencies: `pip install -r requirements.txt`
3. Install frontend dependencies: `npm install`
4. Configure environment variables
5. Run migrations: `python manage.py migrate`
6. Start the development servers

### Configuration
- Set up Google Gemini API key in environment variables
- Configure database settings
- Set up email settings for contact and feedback forms
